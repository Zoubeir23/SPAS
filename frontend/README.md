# ISI Academic System - Frontend

Application frontend React + TypeScript + Vite pour le système d'information académique et analytique.

## Technologies

- **React 18.3+** - Bibliothèque UI
- **TypeScript** - Typage statique
- **Vite** - Build tool
- **Tailwind CSS** - Framework CSS
- **React Router** - Routing
- **Zustand** - Gestion d'état
- **React Hook Form + Zod** - Formulaires et validation
- **Axios** - Client HTTP

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

L'application sera accessible sur `http://localhost:5173`

## Build

```bash
npm run build
```

## Tests

```bash
npm run test
```

## Formatage

```bash
npm run format
```

## Structure du projet

```
frontend/
```
