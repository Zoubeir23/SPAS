========================================
OPTIMISATIONS POSTGRESQL APPLIQUEES
========================================

1. Connection Pooling
   - CONN_MAX_AGE = 600 secondes (10 minutes)
   - Réutilisation connexions pour réduire latence

2. Health Checks
   - CONN_HEALTH_CHECKS = True
   - Vérification automatique connexions

3. Timeouts
   - connect_timeout = 10 secondes
   - statement_timeout = 30000 ms (30 secondes)

4. Transactions
   - ATOMIC_REQUESTS = True
   - Chaque requête HTTP enveloppée dans transaction

5. Extensions PostgreSQL
   - uuid-ossp: Génération UUIDs
   - pg_trgm: Optimisation recherche texte
